This is a wikiscraper for the Bulbapedia wiki.

Example use

Move to directory wikiscraper/wikitools and install the package in your virtual environment
pip install .

You can now view the docs for this package using
python3 -m pydoc wikitools

To use the analysis notebook (analysis/language_analysis.ipynb), run it using an environment with the wikitools package installed.
